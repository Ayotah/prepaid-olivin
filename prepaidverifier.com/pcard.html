Récupère un code prépayé PaysafeCard près de chez toi et paye en ligne avec le code à 16 chiffres. Fiable et sécurisé, sans carte de crédit ni compte bancaire. Il n’a jamais été aussi facile de payer avec de l’argent liquide en ligne.

Il y a deux possibilités pour payer en ligne avec PaysafeCard :

Sans inscription

  • Tu as seulement besoin de ton code à 16 chiffres pour payer
  • Nous n’avons pas besoin de données personnelles
  • Sans compte bancaire ni carte de crédit
  • Frais de mise à disposition à partir du 2ème mois. Tu trouveras tous les détails ici.

Montant max. par transaction : jusqu'à 50 EUR *

Sur certaines boutiques en ligne, une inscription peut être nécessaire pour toutes les transactions.
